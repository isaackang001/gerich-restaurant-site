# Gericht Restaurant
Gericht Restaurant is a small project that utilizes all technologies used below:
- About the restaurant and History
- Drink Menu
- Chef Information
- Dining Video
- Awards
- Photo Gallery
- Contact information and operating hours
- Newsletter

## Technologies Used
- HTML
- CSS
- JavaScript
- React.js
- React Hooks

### System Requirements
- npm

### Set Up Environment
1. Clone the repo
  ```shell
  git clone https://github.com/isaackang001/gerich-restaurant-site.git
  cd gerich-restaurant-site
  ```

2. NPM Install
  ```shell
  npm install 
  ```

3. Start the development server
  ```shell
  npm start
  ```
